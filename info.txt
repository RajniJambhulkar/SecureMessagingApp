Secure Chat (PWA-enabled Real-Time Messaging Application)

Built with ASP.NET Core, Angular, SignalR, and JWT Authentication.
Real-time one-to-one messaging using SignalR.
Typing indicators and online/offline presence tracking.
Message history with pagination.
Responsive design for desktop and mobile devices.
Progressive Web App (PWA) support with installable experience and offline asset caching.
Profile image upload and secure authentication.

What is a PWA?

PWA (Progressive Web App) is a web application that behaves like a native mobile or desktop app while still being built with web technologies such as Angular, React, HTML, CSS, and JavaScript.

Think of it as:

Website + App-like Experience = PWA
